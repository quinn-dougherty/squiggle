# Math

### E

```
Math.e:
```

### LN2

```

```

### LN10


### PI

```
Math.pi:
```

### PHI

```
Math.phi:
```

### TAU

```
Math.tau:
```

Tau is the ratio constant of a circle's circumference to radius, equal to 2 \* pi. 6.283185307179586
