# Number

Squiggle `numbers` are Javascript floats.

Many of the functions below work on lists or pairs of numbers.

### ceil

```
ceil: (number) => number
```

### floor

```
floor: (number) => number
```

### abs

```
abs: (number) => number
```

### round

